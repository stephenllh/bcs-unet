## Getting Started

To get a local copy up and running, follow these simple example steps.
<br/><br/>

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/stephenllh/bcs-unet.git
   ```

1. Change directory
   ```sh
   cd bcs-unet
   ```

2. Install packages
   ```sh
   pip install requirements.txt
   ```
<br/>

<!-- USAGE EXAMPLES -->
## Usage

1. To train, run the training script
   ```sh
   cd ..
   python train.py
   ```
